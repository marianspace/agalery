Pre entrega N3 Abalo
Api web con Python, Django, Html en VSCode

Para ingresar a la web http://127.0.0.1:8000/agalery/ 
El menu incluye 
enlace a inicio
enlace a artistas que devuelve listado de los artistas en base de datos
enlace a obras que devuelv listado de obras en base de datos 
enlace a busqueda de obras por titulo
enlace a ingreso a la plataforma

Para ingresar admin http://127.0.0.1:8000/admin/ 
Las bases de datos entan relacionadas por la variable user y las obras con los artistas.
